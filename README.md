# Resume Bank

A minimal project skeleton for the Resume Bank application.

This repository contains a **FastAPI** backend and a **React + Tailwind CSS** frontend.

## Backend

The backend lives in `backend/` and exposes a simple FastAPI application. Install dependencies with:

```bash
pip install -r requirements.txt
```

Run locally with:

```bash
uvicorn main:app --reload
```

## Frontend

The frontend lives in `frontend/` and is bootstrapped with Vite and Tailwind CSS.
Install dependencies via npm and start the dev server:

```bash
npm install
npm run dev
```

This is only a starting point and does not provide production-ready configuration.
